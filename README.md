A two-player dice game built with JavaScript.

## ðŸŽ® Game Rules
- The game is played by **two players**
- On each turn, a player can:
  - **Roll the dice**
  - **Hold** the current score
- If a player rolls a **1**:
  - the current score is lost
  - the turn switches to the other player
- When a player chooses **Hold**:
  - the current score is added to the total score
  - the turn switches to the opponent
- The **first player to reach 100 points wins the game**
